# Prompt Versioning Simple Python Example

This example demonstrates the concept of prompt versioning, crucial for managing AI applications. It shows how different versions of a prompt, each with a unique tag and description, can be stored and selected. The example simulates using these prompts with a hypothetical LLM to highlight how prompt variations can lead to different desired behaviors.

## Language

`python`

## How to Run

Save the code as `main.py`.
Open a terminal in the same directory and run `python main.py`.
